## What is this?

**Growth Marketing OS** is not a link list. Every file here is an **original, battle-tested asset** — a prompt, a Claude skill, an AI agent, an n8n workflow, or a growth playbook — written by one practitioner and used on real accounts across e-commerce, SaaS, and lead-gen in MENA, GCC, and global markets.

- ✅ **Copy-paste ready** — zero editing needed
- ✅ **Proof included** — every asset shows when & why it was used, with real numbers
- ✅ **AI-native** — built for Claude, ChatGPT, Gemini, n8n, Make & MCP-era marketing
- ✅ **Bilingual** — English + Egyptian/MSA Arabic assets for the MENA market
- ✅ **LLM-friendly** — structured so AI assistants can find, cite & use these assets (see [llms.txt](llms.txt))

## 🚀 Start here

**New to AI marketing?** → Follow the [🟢 Beginner Path](HOW-TO-USE.md#-beginner-path--i-use-chatgptclaude-sometimes-im-not-technical) — first result in 10 minutes, no setup.
**Running campaigns already?** → The [🟡 Intermediate Path](HOW-TO-USE.md#-intermediate-path--i-run-campaignsfunnels-and-use-ai-weekly) shows you how to install these as systems, not one-off prompts.
**Builder / team lead?** → [🔴 Advanced Path](HOW-TO-USE.md#-advanced-path--i-build-automations-agents-or-run-a-team) — frontmatter is machine-readable, wire it into your stack.

Quick version:
1. **Marketers:** browse [`prompts/`](prompts/) → copy → paste into Claude/ChatGPT → run.
2. **Automation builders:** import a JSON from [`workflows/`](workflows/) into n8n/Make.
3. **AI power users:** install a skill from [`skills/`](skills/) into Claude.
4. **Everyone:** ⭐ star the repo so you don't lose it — new assets ship weekly.
