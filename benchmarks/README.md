# 📊 Marketing Benchmarks (Sourced) — Growth Marketing OS by Mahmoud Omar

Market benchmarks across every marketing discipline — **every figure cited to a named study with its sample size and year**. Where no reliable public data exists, the file says so instead of inventing a number. Weak vendor-blog figures are flagged, methodology conflicts are shown, and each file ends with rules for using benchmarks without being misled by them.

> Benchmarks are thresholds for attention, never targets. Use them to find which metric deserves the investigation — then investigate with the matching prompt.
