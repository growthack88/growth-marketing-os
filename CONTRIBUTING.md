# Contributing to Growth Marketing OS

The bar here is **quality over volume**. This is not an awesome-list — every file is an original, tested asset.

## Rules

1. **Use the template.** Copy [`templates/ASSET_TEMPLATE.md`](templates/ASSET_TEMPLATE.md). Fill every frontmatter field.
2. **Proof or it doesn't ship.** The "When I use it" section must describe a real scenario. Generic theory gets rejected.
3. **Copy-paste ready.** The asset block must work with zero editing.
4. **Naming:** kebab-case, keyword-first filenames — `meta-ads-creative-diagnosis.md`, not `prompt1.md`.
5. **One asset per file.** One file = one page = one shareable unit.
6. **Language:** EN, AR, or bilingual — mark it in frontmatter. Arabic contributions are especially welcome.
7. **No affiliate links. No self-promo inside the asset body.** Contributors get credit in the file header (`contributor:` field) and the Hall of Fame.

## Flow

Fork → add your asset under the right folder → open PR → maintainer review (expect edits) → merge → the asset auto-flows into the newsletter & social channels with your credit.
