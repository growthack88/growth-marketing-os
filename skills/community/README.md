# 🌍 Community Skills (curated, MIT-licensed)

These skills are **not original work by Mahmoud Omar** — they're the best-in-class open-source marketing skills, curated into this repo under their original MIT licenses so you get a complete toolkit in one install. Full attribution in every file; original license preserved at [LICENSE-marketingskills.txt](LICENSE-marketingskills.txt).

| Skill | What it does | Source |
|---|---|---|
| [Marketing Psychology](marketing-psychology/SKILL.md) | 70+ mental models & psychological principles applied to marketing decisions | [coreyhaines31/marketingskills](https://github.com/coreyhaines31/marketingskills) (MIT) |
| [A/B Testing](ab-testing/SKILL.md) | Test design, sample-size math, and templates — with reference guides | same |
| [Programmatic SEO](programmatic-seo/SKILL.md) | Scaled page generation done right: patterns, playbooks, quality gates | same |
| [Churn Prevention](churn-prevention/SKILL.md) | Retention systems: cancel-flow patterns, dunning playbook, save offers | same |
| [Onboarding](onboarding/SKILL.md) | Activation-focused onboarding design + experiment library | same |
| [Positioning & Messaging Framework](positioning-messaging-framework/SKILL.md) | Build positioning docs and messaging hierarchies that survive contact with sales | [adam-lagerhausen/b2b-marketing-skills](https://github.com/adam-lagerhausen/b2b-marketing-skills) (MIT) |
| [Launch Tier Framework](launch-tier-framework/SKILL.md) | Tier launches by impact and size the GTM motion accordingly | same |
| [Voice of Customer Synthesis](voice-of-customer-synthesis/SKILL.md) | Turn raw customer research into structured messaging inputs | same |
| [Battlecards](battlecards/SKILL.md) | Competitive battlecards for sales enablement | [matteotitta/genesys-skills](https://github.com/matteotitta/genesys-skills) (MIT) |
| [ICP Research](icp-research/SKILL.md) | Define and research your ideal customer profile systematically | same |
| [Win-Loss Analysis](win-loss/SKILL.md) | Structured win/loss interviews and pattern synthesis | same |

**Why these:** they fill gaps the [original skills](../README.md) don't cover (psychology reference, testing methodology, pSEO, retention ops, onboarding, and the product-marketing/GTM layer: positioning, launches, ICP, battlecards, win-loss) without duplicating them. They compose with this repo's assets — e.g. run [Growth Experiment Designer](../../prompts/growth/growth-experiment-designer.md) to build the backlog, then the A/B Testing skill to size and run each test; use Churn Prevention alongside the [win-back sequence](../../prompts/email/winback-reactivation-sequence.md).
