| Skill | What it does | Source |
|---|---|---|
| [Marketing Psychology](marketing-psychology/SKILL.md) | 70+ mental models & psychological principles applied to marketing decisions | [coreyhaines31/marketingskills](https://github.com/coreyhaines31/marketingskills) (MIT) |
| [A/B Testing](ab-testing/SKILL.md) | Test design, sample-size math, and templates — with reference guides | same |
| [Programmatic SEO](programmatic-seo/SKILL.md) | Scaled page generation done right: patterns, playbooks, quality gates | same |
| [Churn Prevention](churn-prevention/SKILL.md) | Retention systems: cancel-flow patterns, dunning playbook, save offers | same |
| [Onboarding](onboarding/SKILL.md) | Activation-focused onboarding design + experiment library | same |

**Why these five:** they fill gaps the [original skills](../README.md) don't cover (psychology reference, testing methodology, pSEO, retention ops, onboarding) without duplicating them. They compose with this repo's assets — e.g. run [Growth Experiment Designer](../../prompts/growth/growth-experiment-designer.md) to build the backlog, then the A/B Testing skill to size and run each test; use Churn Prevention alongside the [win-back sequence](../../prompts/email/winback-reactivation-sequence.md).
