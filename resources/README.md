Vendored copies of the best openly-licensed marketing resources — **not original work by Mahmoud Omar** — included under their original licenses with full attribution so the toolkit is complete in one clone. Each file carries its source, snapshot date, and license.

**How this composes with the rest of the repo:** the directory answers "which tool?"; this repo's [prompts](../prompts/), [skills](../skills/), and [benchmarks](../benchmarks/) answer "how, and against what numbers." Its own advice agrees with ours: start with prompts, pay for tools only when they add real features beyond a wrapped prompt.

Snapshots are dated — check the source repos for the latest versions, and star them; open-source curation deserves it.
